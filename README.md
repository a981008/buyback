# 港股股票回购查询

查询香港上市公司股票回购数据的工具。

访问：https://a981008.github.io/buyback/

## 功能

- 港股股票搜索及自动补全
- 股票回购数据查询
- 实时显示股价、总市值、总股本
- 历史查询记录
- 跳转到雪球（实时报价）和同花顺（股份回购）

## 技术栈

- Vue 3 + TypeScript
- Tailwind CSS
- Vite

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 部署

推送到 GitHub 后自动通过 GitHub Actions 部署到 GitHub Pages。

## 数据来源

数据来源于同花顺（10jqka.com.cn）接口，仅供参考。
